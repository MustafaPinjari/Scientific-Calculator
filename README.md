# Scientific Calculator üßÆ

The Scientific Calculator is a graphical calculator application built using Tkinter in Python. It provides a user-friendly interface for performing mathematical calculations, including basic arithmetic, trigonometric functions, logarithmic functions, and more.

## Features üåü

- **Standard Mode:** Allows basic arithmetic operations such as addition (+), subtraction (-), multiplication (√ó), and division (/).
- **Scientific Mode:** Provides additional scientific functions including trigonometric functions (sine, cosine, tangent), logarithmic functions (log, log10, log1p, log2), exponentiation (exp), and more.
- **Memory Functions:** Supports memory operations like memory clear (AC), memory recall (MR), and memory addition (M+).
- **Responsive Design:** Adjustable window size with resizable and non-resizable modes.
- **Menu Options:** Includes menu options for switching between Standard and Scientific modes, as well as basic editing functions (cut, copy, paste).
- **Exit Confirmation:** Prompts the user for confirmation before exiting the application.

## How to Run üöÄ

1. Ensure you have Python installed on your system.
2. Copy the provided Python code into a new Python file (e.g., `scientific_calculator.py`).
3. Open a terminal or command prompt.
4. Navigate to the directory where `scientific_calculator.py` is located.
5. Run the following command to execute the program:
   ```bash
   python scientific_calculator.py
   ```
6. The Scientific Calculator window will appear, ready for use.

## Usage üñ•Ô∏è

### Standard Mode:
- Perform basic arithmetic calculations using the number pad and operation buttons.
- Use buttons like AC (All Clear), MR (Memory Recall), and M+ (Memory Addition) for memory operations.

### Scientific Mode:
- Access advanced mathematical functions by switching to the Scientific mode using the menu options.
- Use functions like sin (sine), cos (cosine), tan (tangent), log (logarithm), exp (exponential), etc.

### Copy/Paste and Editing:
- Use the Edit menu (cut, copy, paste) for basic editing operations within the calculator.
- Utilize the scientific functions and memory options for more complex calculations and data storage.
